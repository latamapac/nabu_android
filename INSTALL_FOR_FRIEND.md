## For Your Friend:

### Step 1: Allow Unknown Apps
**Android 8.0+ (Oreo and newer):**
1. Open **Settings**
2. Go to **Apps** â†’ **Special app access**
3. Tap **Install unknown apps**
4. Select the app you'll use to open APK (e.g., Chrome, Files)
5. Enable **Allow from this source**

**Android 7.0 and older:**
1. Open **Settings**
2. Go to **Security**
3. Enable **Unknown Sources**

### Step 2: Install the APK
1. Receive the APK file (via Telegram, Email, etc.)
2. Tap the downloaded file
3. Tap **Install**
4. Open the app!

### Step 3: Create Account
1. Create a local account (no internet needed)
2. Start chatting!
